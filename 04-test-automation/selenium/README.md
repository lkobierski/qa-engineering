# Selenium Test Automation

This folder contains UI test automation projects created with Selenium WebDriver.

## Project overview

### SauceDemo Cart Tests

The `saucedemo-cart-tests` project contains automated UI tests for the SauceDemo cart flow.

The goal of this project is to demonstrate classic browser automation using Selenium WebDriver with JavaScript, Mocha, and the Page Object Model pattern.

Main coverage:

- login before cart tests
- adding a single product to cart
- adding multiple products to cart
- verifying cart item details
- removing a product from cart
- continuing shopping from cart

## Technologies used

- JavaScript
- Selenium WebDriver
- Mocha
- ChromeDriver
- Node.js assert
- Page Object Model

## How to run

Go to the project folder:

```bash
cd saucedemo-cart-tests
```

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npm test
```

## Test results

Test results are displayed in the terminal by Mocha.
