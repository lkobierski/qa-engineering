# QA Engineering Portfolio

This repository contains my QA portfolio projects focused on practical software testing skills and real-world QA workflows.

The goal of this repository is to demonstrate my ability to design, document, execute, and automate tests across different testing areas.

## About

The repository is organized into separate sections, where each section focuses on a specific QA area or testing tool.

## Repository Structure

| Section | Description |
|---|---|
| [01-test-documentation](./01-test-documentation/) | Manual test cases, exploratory testing notes, bug reports and QA documentation |
| [02-api-testing](./02-api-testing/reqres/) | API testing documentation and Postman resources |
| [03-database-testing](./03-database-testing/) | SQL practice and database testing examples |
| [04-test-automation](./04-test-automation/) | API automation and automated tests using Playwright, Cypress and Selenium |
| [05-performance-testing](./05-performance-testing/) | Performance testing examples using JMeter |
| [06-ci-cd](./06-ci-cd/) | CI/CD examples using GitHub Actions |
