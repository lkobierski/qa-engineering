# GitHub Actions

This folder contains GitHub Actions examples used for QA automation workflows.

GitHub Actions is used here to run automated tests directly in GitHub after repository changes.

---

## Projects

| Project | Description |
|---|---|
| [playwright-tests-workflow](./playwright-tests-workflow/) | Workflow for running Playwright UI tests |

---

## Covered Areas

- workflow configuration
- Node.js setup
- dependency installation
- Playwright browser installation
- automated test execution
- CI result review

---

## Notes

The examples are intentionally simple and focused on QA portfolio purposes.
